NPayE System

This was a project I started back in November 2018. I was the sole creator/developer of the project.

The system was used to buy and sell tokens of Electricity and Telecoms.

Pin based Recharges- Customer can approach an agent and buy a telecom voucher.

Electricity-Customer can approach an agent and buy a token or pay his electricity bill .

I worked on this system for 2 years before Launching it in Mozambique.

Technologies:
  • - Php
  • - Mysql
  • - Android
  • - Google Maps API
