NPayE System

This was a project I started back in November 2018. I was the sole creator/developer of the project.

The system was used to buy and sell tokens of Electricity and Telecoms,CRUD for Merchants, Products, Commissions, Transaction Reports.

Reports would be generated by the system in PDF/EXCEL form and automatically sent to the client along with an interface for customers to be able to login and view.

Customers had the ability to view all their data with a dynamic search form and even map the merchants physical location.

I worked on this system for 2 years before Launching it in Mozambique.

Technologies:
  • - Php
  • - Mysql
  • - Google Maps API
  • - JS Charts
  • - FPDF
